# Beyond Population: Understanding the Socioeconomic Drivers of UPI Adoption in India

## Overview

This project investigates the socioeconomic factors influencing UPI adoption across Indian states using statistical analysis and machine learning models.

## Objectives

- Analyze UPI adoption trends
- Study socioeconomic drivers
- Build predictive machine learning models
- Evaluate model performance

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Matplotlib

## Results

- XGBoost R² Score: 0.967
- RMSE: 0.88
